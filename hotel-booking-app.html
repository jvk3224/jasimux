Hotel Booking System

Role: UX/UI Designer

The Hotel Booking System is a user-centric platform designed to provide a seamless hotel booking experience. The project showcases my ability to design intuitive and engaging interfaces for a modern booking platform, emphasizing usability, security, and personalization.

Key Features Designed

User Research
  • Centralized view of bookings, recommended hotels, and personalized deals.
  • Easy navigation with quick access to essential features.
    •
    Login and Sign-Up
  • User-friendly login and sign-up process.
  • Secure authentication methods for better user trust.
    •
    Face Authentication
  • Innovative security feature allowing users to log in securely using face recognition.
    •
    Profile Management
  • A comprehensive user profile section for managing personal information, preferences, and booking history.
    •
    Rewards Page
  • Dedicated page showcasing loyalty points and redeemable rewards, enhancing user engagement and retention.
    •

    Challenges & Solutions

    • Challenge: Ensuring seamless navigation across multiple pages for diverse user needs

    • Solution: Designed a clean and intuitive layout with consistent visual hierarchy.
    • Challenge: Implementing face authentication with a focus on user security.

    • Solution:Created a straightforward, visually appealing flow for the feature, ensuring clarity at every step.

    Outcomes

    The Hotel Booking System design highlights the importance of combining usability with innovative features like face authentication and rewards management. This design can be implemented as a fully functional system to enhance user experience in the hospitality sector.
